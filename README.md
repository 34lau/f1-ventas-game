# F1 Sales Grand Prix - Sales Gamification Dashboard

**F1 Sales Grand Prix** es una herramienta de visualización dinámica diseñada para transformar el entorno de un Call Center en una parrilla de salida de la Fórmula 1. Este videojuego de escritorio permite al supervisor de ventas registrar resultados en tiempo real, fomentando la competitividad y la motivación del equipo mediante estímulos visuales y auditivos de alto impacto.

## Características Principales
- **Parrilla de Resultados (Live Leaderboard):** Clasificación en tiempo real de 6 escuderías (Ferrari, Red Bull, Mercedes, McLaren, Williams, Haas).
- **Dinámica de Carrera:** Cada venta registrada activa una vuelta rápida del monoplaza en el circuito con efectos de sonido y partículas.
- **Panel de Control del Supervisor:** Interfaz centralizada para configurar nombres de pilotos, metas individuales y meta grupal mensual.
- **Sistema de Dopamina:** Recompensas visuales por hitos alcanzados, uso de DRS para adelantamientos y efectos de "Purple Sector".
- **Persistencia de Datos:** Utiliza `LocalStorage` para garantizar que el progreso de la carrera no se pierda al cerrar el navegador o refrescar la página.

## Especificaciones Técnicas
- **Lenguajes:** HTML5, CSS3 (Custom Properties & Animations), JavaScript (ES6+).
- **Motor Visual:** Canvas API / GSAP para animaciones fluidas de los monoplazas.
- **Diseño:** Estética Dark Mode basada en fibra de carbono y neones dorados.
- **Despliegue:** Optimizado para **GitHub Pages**.

## Instrucciones de Uso
1. Abre el link del juego en Google Chrome.
2. Presiona la tecla **'S'** para abrir el panel de configuración.
3. Registra a tus 6 vendedores con sus respectivas metas.
4. ¡Inicia la carrera! Cada vez que un agente venda, regístralo para ver cómo escala posiciones en la pista.
